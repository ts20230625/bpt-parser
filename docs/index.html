BPT Parser

嵌入式 SoC 二进制参数表 (Binary Parameter Table) 解析与编辑工具，支持 Intel HEX 和 BIN 格式

下载 Windows 版

功能特性

文件导入导出

支持导入 Intel HEX (.hex) 和原始二进制 (.bin) 文件，自动识别格式。支持另存为 HEX 或 BIN 格式导出。

HEX BIN

完整字段解析

解析 BPT 全部字段：Header、8 个 IIB 镜像信息块、RCP 根证书包、签名、Trailer。枚举字段显示中文名称。

BPT Header IIB RCP

字段编辑

支持修改所有可编辑字段（整数、枚举、位域、字节序列），修改后实时同步更新 Hex 视图和字段树。

实时同步

Hex 原始数据视图

右侧面板显示完整 Hex 转储，点击字段自动高亮对应字节区域并滚动到居中位置。支持修改基地址偏移。

高亮定位

自动计算

修改任意字段后自动重算 CRC32（覆盖 0x0~0xFEB）和 PSN 反码（0xFFFFFFFF - PSN），无需手动操作。

CRC32 PSN

撤销与恢复

支持撤销所有改动（一键恢复到导入时的原始数据），也支持单个字段独立恢复（仅被修改的字段显示恢复按钮）。

单字段恢复 全部撤销

界面布局

BPT 数据结构

结构偏移大小说明
BPT Header0x00032 字节标签、版本、摘要算法、密钥选择
IIB #0 ~ #70x020各 124 字节8 个镜像信息块：设备ID、镜像类型、加载地址、入口点、哈希
RCP0x4001044 字节根证书包：ROT ID、公钥类型（RSA/ECDSA/SM2）、公钥数据
Signature0x814512 字节数字签名
Reserved0xA14512 + 984 字节保留区域
BPT Trailer0xFEC20 字节CRC32、包序列号 (PSN)、PSN 反码

支持的枚举字段

摘要算法

SHA256 (0x03) · SHA512 (0x05) · SM3 (0x06)

镜像类型

Normal (0x00) · SCB (0x01) · HSMFW (0x0E)

目标核心

Cluster0-Core0 (0x00) · SE (0x03) · LP (0x04)

公钥类型

RSA1024/2048/3072/4096 · ECDSA P256/P384 · SM2
